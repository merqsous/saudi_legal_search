# Albaheth Developer Brand Kit

A framework-free, developer-ready bilingual brand and landing-page starter for **Albaheth / الباحث**.

## Preview locally

Open `index.html` directly, or run a local server:

```bash
python3 -m http.server 8080
```

Then visit `http://localhost:8080`.

## Main files

- `index.html` — English landing page
- `arabic.html` — Arabic RTL landing page
- `css/styles.css` — complete page styles
- `css/brand-tokens.css` — reusable CSS variables
- `design-tokens.json` — tool-neutral design tokens
- `AI-CODING-CONTEXT.md` — instructions for AI coding tools
- `site.webmanifest` — PWA/browser metadata
- `assets/logos/` — Arabic and English logos
- `assets/icons/` — favicon and application mark
- `assets/social/` — social and Open Graph images

## Important note

The Arabic calligraphy is supplied as a raster PNG generated from the approved visual. The English logo and icon are editable SVG files. For trademark registration, large-format printing, or exact professional reproduction, have a designer redraw the approved Arabic calligraphy as a true vector file.
